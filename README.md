# 🧬 Medical-DataScience

Machine Learning and Deep Learning approaches for **lung cancer detection from medical images**. This project combines **dimensionality reduction, statistical analysis, and neural networks** to build efficient and accurate diagnostic models.

---

## 📌 Project Overview

Early detection of lung cancer can significantly improve patient outcomes. This repository explores the use of **Principal Component Analysis (PCA)** and **Convolutional Neural Networks (CNNs)** to classify medical images and predict whether a patient has lung cancer.  
The final model achieves **96% accuracy on the test dataset**.

---

## 🔍 Key Components

### Dimensionality Reduction (PCA)
- Applied **Principal Component Analysis** to reduce high-dimensional image data.
- Improved runtime and efficiency of machine learning algorithms.

### Deep Learning Classification
- Implemented **Convolutional Neural Networks (CNNs)** for image classification.
- Automatically learned spatial features from lung scan images.

### Model Validation
- Used **k-fold cross-validation** to evaluate model robustness.
- Performed **hyperparameter tuning** to optimise performance.

### Statistical Analysis
- Generated **Scree Plots** to determine optimal principal components.
- Conducted **Analysis of Variance (ANOVA)** for feature evaluation.
